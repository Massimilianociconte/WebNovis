1. Componenti UI (Copy-Paste & Frameworks)
Risorse per elementi d'interfaccia pronti all'uso (bottoni, card, form) da integrare nel codice.
Generalisti & Tailwind (Copia-Incolla)
Uiverse.io: Elementi UI open-source creati dalla community (HTML/CSS, Tailwind).
HyperUI: Componenti per marketing ed e-commerce (HTML + Tailwind).
Float UI: Sezioni di pagina complete per React, Vue, Svelte e HTML.
Tailblocks: Sezioni pronte all'uso per landing page rapide (Tailwind).
Meraki UI: Componenti con supporto RTL e dark mode (Tailwind).
WickedBlocks: Blocchi pre-costruiti per siti di marketing.
FreeFrontend: Collezioni curate di esempi creativi da CodePen.
CodeMyUI: Snippet HTML/CSS/JS con anteprime GIF.
Ecosistema React (Focus su Shadcn/ui)
shadcn/ui: Primitives accessibili (Radix) stilizzate con Tailwind; filosofia "possiedi il tuo codice".
Origin UI: Estensione di shadcn con 400+ componenti.
Aceternity UI: Componenti shadcn con animazioni Framer Motion.
Magic UI: Componenti animati ad alto impatto visivo.
Mantine: Sistema completo con 120+ componenti e hook.
Ant Design: Componenti di livello enterprise.
Radix UI: Primitive non stilizzate focalizzate sull'accessibilità (base di shadcn).
Tremor: Componenti specializzati per dashboard e grafici.
Librerie Tailwind CSS
DaisyUI: Aggiunge classi semantiche e temi a Tailwind (no JS).
Flowbite: Vasta libreria con adattatori per i principali framework JS.
Preline UI: Grande design system Figma con plugin Tailwind.
FlyonUI: Classi semantiche combinate con plugin JS.
Sailboat UI: Componenti interattivi basati su Tailwind e Alpine.js.
Ecosistema Vue
Vuetify: Implementazione del Material Design per Vue.
PrimeVue: Libreria completa e agnostica rispetto al design.
Nuxt UI: Componenti accessibili per l'ecosistema Nuxt.
shadcn-vue: Porting della filosofia shadcn per Vue.
Naive UI: Sistema basato su TypeScript con 80+ componenti.
CSS-Only & Stili Particolari
Pico.css: Stile automatico per elementi HTML nativi (senza classi).
NES.css: Interfacce in stile retro 8-bit.
Neobrutalism.dev: Componenti in stile neobrutalista (basato su shadcn).
Bulma: Framework puramente CSS senza dipendenze JS.
2. Icone
Librerie vettoriali per ogni stile visivo.
I "Grandi Classici"
Lucide: Fork di Feather Icons, consistente e supportata ovunque.
Tabler Icons: Vasta collezione di icone outline di alta qualità.
Phosphor Icons: Flessibile (6 pesi diversi, incluso duotone).
Heroicons: Set del team Tailwind, semplice e pixel-perfect.
Remix Icon: Ogni icona disponibile in stile outline e filled.
Bootstrap Icons: Ottimizzate per l'ecosistema Bootstrap.
Set Specializzati & Gemme Nascoste
Iconoir: Nessun livello premium, open source.
MingCute: Icone artigianali di alta qualità.
css.gg: Icone realizzate in puro CSS (no immagini/font).
Myna UI Icons: Vasto set con licenza MIT.
Atlas Icons: Spessore del tratto regolabile via browser.
Simple Icons: Loghi di brand e marchi (SVG).
Pixelarticons: Stile pixel art retrò.
Radix Icons: Icone 15x15 ideali per tool di sviluppo.
Octicons: Set ufficiale di GitHub.
Icone Animate & Aggregatori
pqoqubbw/icons: Icone animate con Framer Motion.
useAnimations: Icone lineari animate (Lottie).
Lucide Animated: Versioni animate delle icone Lucide.
Iconify: API unificata per 200k+ icone da set diversi.
Icones: Interfaccia di ricerca visuale per Iconify.
React Icons: Bundle di icone per React (con tree-shaking).
SVG Repo: Enorme repository di SVG.

LottieFiles: Animazioni JSON vettoriali scalabili.
